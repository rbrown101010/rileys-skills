# YouTube Channel Latest Summary

Use this skill when Riley asks for the latest videos from a YouTube channel,
recent uploads from a creator, or a compact summary of a channel's newest
content.

## Workflow

1. Identify the channel URL, handle, or creator name.
2. Use available YouTube or transcript tooling to find the most recent uploads.
3. Prefer transcript-grounded summaries when transcripts are available.
4. Keep the output compact unless Riley asks for deeper research.

## Output

- Include video title, URL, publish date when available, and a concise summary.
- Note when a transcript was unavailable or when a summary is metadata-only.
